# Pet Shop Project

Bem-vindo ao repositório do **Pet Shop Project**! 🐾

Este é um projeto de demonstração de um sistema/site para um Pet Shop, desenvolvido como um exemplo de aplicação web.

## 🚀 Funcionalidades

- Exibição de serviços oferecidos (Banho, Tosa, Consulta Veterinária, etc.)
- Catálogo de produtos para pets
- Interface responsiva e amigável

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📦 Como executar o projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/petshop.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd petshop
   ```
3. Abra o arquivo `index.html` no seu navegador de preferência.
